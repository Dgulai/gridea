从零学Java-00 Java,JDK,JRE,JVM

2020-08-15

Java特性和优势

  • 简单性
  • 面向对象
  • 可移植性
  • 高性能
  • 分布式
  • 动态性
  • 多线程
  • 安全性
  • 健壮性

Java 三大版本

Write Once Run Anywhere

  • JavaSE: 标准版(桌面程序，控制台开发...)
  • JavaME: 嵌入式开发(手机，小家电...)
  • JavaEE: E企业级开发(web端，服务器开发...)

JDK JRE JVM (往右包含)

  • JDK: Java Development Kit
  • JRE: Java Runtime Environment
  • JVM: JAVA Virtual Machine

Java开发环境

  1. 下载JDK并安装
  2. 配置环境变量 (JAVA_HOME\bin;JAVA_HOME\jre\bin)

Java程序运行机制

  • 编译型(javac编译成class)
  • 解释型(在JVM中运行class解释)
